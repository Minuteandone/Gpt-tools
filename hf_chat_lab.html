Find glitch tokens

This tries to inspect the input embedding matrix, computes the centroid, then shows the closest and farthest tokens by Euclidean distance. It works best with models whose embeddings are accessible in Transformers.js.

Fine-tune helper

Real fine-tuning of chat models is generally not practical inside plain browser HTML 🫠, especially for anything beyond tiny experiments. So this tab builds a downloadable training package for LoRA/SFT fine-tuning in Python, based on either an uploaded dataset file or a Hugging Face dataset name. It also lets you preview examples and generate a test script.
